## üöÄ Features

* **Responsive Design:** Works seamlessly on different screen sizes and orientations (Portrait & Landscape).
* **Adaptive UI:** Platform-specific adjustments for iOS and Android components.
* **Game Logic:** Implements a recursive binary-search-like algorithm to guess the number efficiently.
* **Custom Components:** Built using reusable components with advanced styling techniques (Style Merging).
* **Safe Area Views:** Optimized for modern devices with notches (iPhone X+).

## üíª Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Buraksyn0/GuessMyNumber.git](https://github.com/Buraksyn0/GuessMyNumber.git)
    ```

2.  **Navigate to project directory**
    ```bash
    cd GuessMyNumber
    ```

3.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Start the app**
    ```bash
    npx expo start
    ```
